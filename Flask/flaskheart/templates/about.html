About

Heart Disease Predictor is a simple web application that uses the Flask framework to provide a heart disease risk prediction service. The application uses an API developed by Eduard Olteanu to predict the risk of heart disease based on user input. Users can enter their personal information and receive a risk percentage, as well as view their prediction history.

The model used to predict are :

  • Logistic Regression
  • Random Forest
  • Support Vector Machine
  • Decision Tree Classifier
